言語：python
エディター：VSCode

実行手順
同一ファイル上にdata.txtというファイルを用意.
data.txtは各行に1つのデータを記述する.
dictionary.pyをpython3系で実行.
引数として次のように記述する．
$ python3 dictionary.py {ユーザー名} {データファイル名} {単語ID(省略可)}

コマンドライン上での実行例
#############################
$ python3 dictionary.py knishida dictionary_data.txt 1
ユーザー名: knishida
1: 上手 じょうず
#############################

dictionary_data.txtの入力フォーマット
{見出し語1} {読み仮名1}
{見出し語2} {読み仮名2}
...
入力例
#############################
上手 じょうず
一時 いちじ
市場 しじょう 
#############################
